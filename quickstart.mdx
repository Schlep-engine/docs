# Quickstart Guide

Get Schlep-engine up and running in just a few minutes.

## Prerequisites

- Go 1.23 or higher
- Docker (optional, recommended)
- PostgreSQL 14+ (for multi-tenancy)

## Quick Start with Docker

The fastest way to get started:

```bash
# Start Schlep-engine with mock providers
docker-compose up

# Test the API
curl http://localhost:8080/v1/health
```

## Manual Setup

### Step 1: Clone and Build

```bash
# Clone repository
git clone https://github.com/Schlep-engine/schlep-engine.git
cd schlep-engine

# Install dependencies
go mod download

# Build the server
go build -o schlep-engine-api cmd/schlep-engine-api/main.go
```

### Step 2: Configure Environment

Create a `.env` file:

```bash
# Server
PORT=8080
ENV=development

# Provider Mode (mock, benchmark, or real)
PROVIDER_MODE=mock

# Optimizer Mode
OPTIMIZER_MODE=go
```

### Step 3: Start the Server

```bash
# Run with mock providers (no API keys needed)
export PROVIDER_MODE=mock
./schlep-engine-api
```

### Step 4: Make Your First Request

```bash
curl -X POST http://localhost:8080/v1/infer \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4",
    "messages": [{
      "role": "user",
      "content": "Hello, World!"
    }],
    "max_tokens": 100
  }'
```

## Using Real Providers

To use actual OpenAI or Anthropic APIs:

```bash
# Set environment variables
export PROVIDER_MODE=real
export OPENAI_API_KEY=sk-proj-...
export ANTHROPIC_API_KEY=sk-ant-...

# Restart server
./schlep-engine-api
```

## Enable Thompson Sampling

Activate the Rust optimizer for intelligent provider selection:

```bash
# Build Rust kernel
cd rust-core/rust_kernel
cargo build --release
cd ../..

# Enable Rust mode
export OPTIMIZER_MODE=rust
export OPTIMIZER_SAMPLE_RATE=1.0

# Restart server
./schlep-engine-api
```
